# Estudos de Lógica de Programação

Este repositório documenta minha evolução nos estudos de lógica de programação, com foco em construção de raciocínio lógico, algoritmos e fundamentos essenciais para desenvolvimento de software.

## Objetivo

Consolidar os conceitos básicos da programação por meio da resolução prática de exercícios clássicos de lógica, criando uma base sólida para linguagens como Python, Java, C e outras.

## Conteúdos abordados

- Algoritmos e pseudocódigo
- Variáveis e atribuições
- Operadores aritméticos e lógicos
- Estruturas condicionais
- Estruturas de repetição
- Vetores e matrizes
- Teste de mesa e análise de algoritmos

## Metodologia

- Resolução manual dos exercícios
- Teste de mesa
- Versionamento contínuo com commits frequentes
- Organização progressiva dos conteúdos

## Observação

Este repositório está em constante atualização conforme avanço nos estudos.
